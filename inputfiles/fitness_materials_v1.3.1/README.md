# Fitness Materials Database v1.3.1

## Inhoud
- materials.json → materialen
- weights.json → gewichtsklassen (realistisch)
- material_weights.json → koppeling + loadType

## LoadType uitleg
- single → 1 gewicht (kettlebell)
- pair → 2 gewichten (dumbbells)
- barbell → stang + gewichten

## Gebruik
- Combineer materials + weights via material_weights
- Bereken total load afhankelijk van loadType
